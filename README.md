# Image2LMDB
Convert image folder to lmdb, adapted from https://github.com/Lyken17/Efficient-PyTorch

```
├── img

│   │       └── 100009.jpg
│   
│   
├── main.py
├── README.md
└── requirements.txt
```

## Convert image folder to lmdb
```python
python folder2lmdb.py img
````

## Test it
```python
python main.py img/train.lmdb
```
